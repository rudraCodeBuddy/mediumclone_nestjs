## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Practice project implementing backend for : "https://realworld-docs.netlify.app/docs/specs/backend-specs/introduction"

## Installation

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
